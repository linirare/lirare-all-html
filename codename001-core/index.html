按正式项目口径整合对战、天梯、角色、商店与经济四个系统。清理非正式说明、无关实现描述和旧版对照内容，统一跨系统规则、入口关系、奖励归属和版本范围。

一总口径与系统关系

承接主页策划案最终口径：底部导航固定为商店 / 功能牌 / 对战 / 角色 / 排行；中间"对战"为主入口；开始对战直接匹配；功能牌必须配置；聊天首发；赛季/通行证/公会/活动商店后续开放。

二一致性审计结论

本次优化重点不是大改数值，而是统一正式口径、删除无关内容、明确跨系统边界，集中约束容易冲突的规则。

非正式内容
各文档存在非正式对照、背景说明等。
→ 已删除或改写为正式规则。
底部导航口径
主页/对战/排行/天梯混用。
→ 统一为商店/功能牌/对战/角色/排行；天梯归入"排行"。
开始对战
可能出"模式选择页"和"直接匹配"双口径。
→ 统一为点击后直接匹配，不先进模式选择。
功能牌配置
"默认牌兜底" vs "必须配置"双口径。
→ 统一为必须配置 2 张，不配置不进匹配。
PvE 是否计天梯
AI 兜底是否计入段位。
→ 首发计入，后续可拆分练习与正式。
PvP 公平性
角色加成/段位倍率可能被误读为影响战斗。
→ 角色被动不影响 PvP；段位倍率只影响结算奖励。
宝箱来源
对战奖励与商店宝箱冲突。
→ PvP 首发不发宝箱进度；PvE 胜+2，平/负+1。
赛季/通行证
天梯和商店可能提前引入。
→ 首发不做赛季重置/通行证/限时礼包/活动商店。

2.1 已明确但不阻塞首发的待确认事项

事项当前建议
PvE 长期是否计天梯首发计入；当 PvP 在线规模稳定后可拆分 AI 练习与正式。
段位倍率作用范围仅作用于对战基础金币/钻石，不影响任务/签到/邮件/宝箱/角色加成。
宝箱是否加入 PvP首发不加，后续可做 PvP 胜场或段位宝箱。
外装系统首发范围称号/头像框可先规划，不做首发 P0。
⚠ 审计发现：文档中对"金币"和"钻石"的用途描述有轻微不一致——1.3 依赖矩阵说"金币用于角色升级"，6.2 说"金币用于角色升级"，但 3.6 结算表又给出了金币/钻石奖励。实际钻石也可用于角色招募（6.4-6.5），建议明确"钻石是对战+日常+抽卡共用货币，金币是角色升级专用货币"。

三对战系统

核心玩法入口，负责从"开始对战"到"整场结算"的完整闭环。首发采用一键匹配路径。

3.1 对战模式

模式对手入口奖励/天梯定位
PvP 天梯匹配真人开始对战后优先匹配正式奖励 + 天梯变化核心竞争
AI 对战AIPvP 超时兜底正式奖励 + 可计入天梯保证随时可玩
教学AI/固定流程新号引导入口无奖励，不计天梯理解规则与功能牌

3.2 核心战斗规则

  • 5 局 3 胜制，双方初始 3 点生命。
  • 每小局比较双方由出牌 + 3 张固定公共牌组成的最优 5 张牌型。
  • 每场对战开始时发出 3 张公共牌，整场固定。
  • 赢家使对手 -1 生命；生命归零或先赢 3 小局，对战结束。
  • 打满 5 局按小局胜场判定；胜场相同为平局。
小局需出牌数
第 1-3 局3 张
第 4 局4 张
第 5 局5 张

3.3 匹配流程

  1. 点击"开始对战" → 校验功能牌配置和库存。
  2. 进入匹配中界面 → 优先匹配 PvP。
  3. 匹配超时 → 进入 AI 对战。
  4. 对战创建成功 → 扣除带入功能牌库存。

3.4 功能牌使用规则

规则说明
带入数量每场 2 张。
库存消耗对战创建成功时各扣 1 张；取消/教学/未创建不扣。
使用次数每回合最多用 2 张；每张牌每场只能用 1 次。
跨小局重置回合次数，不重置已用功能牌。

3.5 结算与奖励

正式奖励、段位变化、任务进度均以结算结果为准。段位倍率仅作用于对战基础金币/钻石。

AI 胜利200 × 段位倍率50 × 段位倍率+2
AI 平局120 × 段位倍率25 × 段位倍率+1

3.6 UI 与异常处理

  • 对局中隐藏底部导航与非必要入口。
  • 手牌选中需有明确浮起/高亮效果。
  • PvP 出牌超时由服务端代打，客户端只做倒计时展示。
  • PvP 掉线给 15 秒 重连宽限；超时视为离场/投降。
  • 重连后恢复当前对局状态，不重复发奖。

四天梯系统

承接正式对战后的长期成长反馈，向主页、结算页和排行页提供段位、星数/积分、排行展示。首发不做赛季重置，不引入通行证。

4.1 段位结构

大段小段星数/积分升降规则定位
新手局—3 场保护打满 3 场进青铜Ⅲ缓冲区
青铜Ⅲ/Ⅱ/Ⅰ每小段 3 星胜+1，败不掉星入门成长
白银Ⅲ/Ⅱ/Ⅰ每小段 3 星胜+1，败-1基础竞争
黄金Ⅲ/Ⅱ/Ⅰ每小段 4 星胜+1，败-1；可连胜奖励主体成长
铂金Ⅲ/Ⅱ/Ⅰ每小段 4 星胜+1，败-1；无连胜高压过渡
大师—1000 分起胜+25，负-15；1500 进传奇高段竞争
传奇—1500 分+按积分排行；首发可不降段荣誉段

4.2 升降段与保护

  • 满星胜利 → 晋级下一小段/大段，星数重置为 0。
  • 白银+ 在 0 星失败 → 降小段；大段最低小段无保护则降大段。
  • 刚升入白银/黄金/铂金 → 获 1 次大段保护，优先消耗。
  • 青铜永久不掉星，不回新手局。
条件奖励/变化
普通胜利+1 星；大师+ +25 分
普通失败白银+ -1 星；大师+ -15 分
平局星数/积分不变
青铜失败不掉星
青铜/白银/黄金 3 连胜额外 +1 星
铂金+ 连胜无额外星数

4.3 天梯计入范围

模式计入天梯说明
PvP是正式竞争，影响段位和排行。
AI首发计入低在线时保证成长；后续可拆分。
教学否不计入，不发奖励。
练习/测试否后续开放时不影响段位。

4.4 排行页

归属底部"排行"入口。展示段位、星数/积分、保护/连胜提示、战绩摘要、排行榜列表和当前玩家排名。不建议首发强红点。

排序：大段 → 小段 → 星数/积分 → 胜率/总胜场/更新时间。

4.5 天梯输出

输出使用方用途
段位/小段/星数主页、排行、结算身份展示和成长反馈
积分大师/传奇排行高段竞争排序
段位倍率对战结算影响基础金币/钻石
保护/连胜标记结算页说明星数变化来源
