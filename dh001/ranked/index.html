代号 001 · 天梯系统策划案

📄 v1.0 📅 2026-07-07 🏷️ 系统策划 / 一致性优化 🏆 天梯
天梯系统承接正式对战后的长期成长反馈，向主页、结算页和排行页提供段位、星数/积分、排行榜和荣誉展示。首发不做赛季重置，也不引入通行证。

一系统定位

  • 入口：底部"排行"入口进入；结算页可跳转查看天梯变化。
  • 输出：段位、小段、星数/积分、排行榜、保护与连胜反馈。
  • 展示：青铜至铂金展示大段+小段+星数；大师及以上展示积分。

二段位结构

大段小段星数/积分主要规则定位
新手局无3 场人机保护局统一开战入口下 3 场人机对战；正常获奖励，不进排行榜缓冲区
青铜Ⅲ / Ⅱ / Ⅰ每小段 3 星胜利 +1 星，失败不掉星入门成长段
白银Ⅲ / Ⅱ / Ⅰ每小段 3 星胜利 +1 星，失败 -1 星基础竞争段
黄金Ⅲ / Ⅱ / Ⅰ每小段 4 星胜 +1，败 -1；可触发连胜奖励主体成长段
铂金Ⅲ / Ⅱ / Ⅰ每小段 4 星胜 +1，败 -1；无连胜奖励高压过渡段
大师无1000 分起胜 +25，负 -15，1500 分进传奇高段竞争
传奇无1500 分以上按积分排行；首发可不做降段荣誉段

三升降段与保护

  • 青铜至铂金采用"小段 + 星数"；数字越小小段越高，Ⅰ 为该大段最高小段。
  • 满星胜利时立即晋级到下一个小段或大段，星数重置为 0。
  • 白银及以上在 0 星失败会降小段；若处于大段最低小段且无保护，则降到上一大段最高小段。
  • 刚升入白银 / 黄金 / 铂金时获得 1 次大段保护；在最低小段 0 星失败优先消耗保护。
  • 青铜永久不掉星，不会降回新手局。

3.1 跨段位过渡

过渡规则
新手局→青铜打满3场人机保护局自动进入青铜Ⅲ 0/3星
铂金Ⅰ满星→大师晋级大师，初始积分1000
大师→铂金大师积分低于900时掉回铂金Ⅰ3星
大师1000→传奇1500积分达到1500晋级传奇
传奇→大师首发不降段，低于1500仍保留传奇标识

3.2 星数变化表

条件奖励 / 变化
普通胜利+1 星；大师及以上 +25 分
普通失败白银及以上 -1 星；大师及以上 -15 分
青铜失败不掉星
青铜/白银/黄金 3 连胜后胜利额外 +1 星
3 连败后下一场败方不消耗功能牌（仅 1 次，触发后重置连败计数）
铂金及以上连胜无额外星数奖励

四天梯计入范围

模式是否计入天梯说明
PvP 天梯匹配是正式竞争模式，影响段位和排行。
人机兜底局天梯星数/积分与 PvP 同规则；奖励按人机兜底规则（宝箱仅人机兜底积累）后台记录对手类型用于分析。
教学否不计入天梯，不发正式奖励。
练习 / 测试否后续开放时不影响段位。

五排行页结构

天梯归属底部"排行"入口。页面应展示：

  • 当前段位、星数 / 积分
  • 保护 / 连胜提示
  • 战绩摘要
  • 排行榜列表 + 当前玩家排名

排序维度：大段高者优先 → 同大段小段高者优先 → 星数/积分 → 胜率/总胜场/最近更新时间。

排行榜不建议首发强红点，避免与对战主路径竞争。

5.1 天梯输出给其他系统

输出使用方展示 / 用途
当前段位 / 小段 / 星数主页、排行页、结算页身份展示和成长反馈。
积分大师 / 传奇段排行高段竞争排序。
段位倍率对战结算仅影响对战基础金币（钻石固定 1.0）。
保护 / 连胜标记结算页说明本场星数变化来源。
🔗 跨系统依赖：天梯接收对战结算输出的胜负结果；段位倍率输出给对战结算影响金币奖励系数（钻石固定）；段位数据展示给主页、排行页和结算页。
