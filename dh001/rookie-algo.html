二、公共牌质量评分（先发，不依赖手牌）

3 张公共牌自身质量。越高 = 越容易出现大牌型（对所有人公平的好公共牌）。

检测项条件加分
有对子3张中任意点数出现≥2次+25
有三条3张中任意点数出现≥3次+50
同花色≥2张同花色+20
全同花色3张同花色+40（覆盖+20）
顺子基础去重后最大-最小≤4 且至少3个不同点数+30
含 A3张中有 A+15
含 K/Q每张+5

满分约 50+40+30+15+10 = 145。随机发3张公共牌的评分中位数约20-40。

三、手牌评分（后发，依赖公共牌）

分两部分：手牌自身质量 + 手牌与公共牌的匹配度。

手牌自身质量（7张）：

检测项条件加分
有对子任意点数出现≥2次+15
有两对两个不同点数各≥2次+25
有三条任意点数出现≥3次+35
同花趋势某花色≥4张+20
同花趋势强某花色≥5张+30
顺子趋势去重后最大-最小≤5且至少5个不同点数+15
高牌 A/K/Q每张+3

手牌与公共牌匹配度（手牌7张 vs 公共牌3张）：

匹配项说明加分
公共牌花色在手牌中高频公共牌中最多的花色，在手牌中也≥3张+20
凑四条公共牌有对子(X,X)，手牌也有X+30
凑三条公共牌有单张X，手牌有X,X+25
凑对子公共牌有单张X，手牌也有X+10
顺子延伸公共牌中某张与手牌最长连续段相邻+15
公共牌高牌在手牌中有同花色公共牌A/K/Q，手牌中有同花色牌+10/张

手牌自身质量满分约 35+30+15+21=101。匹配度满分约 20+30+25+10+15+30=130。两项叠加满分约 230。

四、补牌友善（依赖手牌+公共牌）

需要补 N 张→牌堆顶多看 N×3 张→给每张打分→取最高 N 张。

匹配项条件加分
与公共牌同花色公共牌中已有该花色的牌+5
与手牌同花色（高频）手牌中该花色≥3张+4
成对（手牌已有）手牌中已有该点数+3
成三条（手牌已有2张）手牌中该点数已有2张+8
顺子延伸与手牌最长连续段相邻+2
与公共牌成对公共牌中已有该点数+5

牌堆不够 N×3→有多少看多少。

五、AI 手牌弱化

仅前10场。模拟3批，取评分最低的给AI。评分标准和玩家完全一样（手牌自身质量+与公共牌匹配度）。

AI弱化叠加在AI难度系统之上，互不冲突。

六、安全阀

每场结算后更新近5场滑动窗口胜率。超过85%→干预概率降一档（100→70→40→15→0）。降了不回弹。

七、触发时序

对局创建 │ ├─ player.matchCount = N → 查衰减表 → 干预概率 P ├─ roll 随机数 R │ R > P → 走正常发牌 │ R ≤ P → 走优化发牌： │ │ ① 发公共牌（3批模拟，取最高分） │ ② 发玩家手牌（3批模拟，自身分+匹配公共牌分=选最高） │ ③ 发 AI 手牌（3批模拟，取最低分，仅前10场） │ ├─ 进入对战（5局3胜正常进行） │ ├─ 每回合补牌时： │ 查衰减表 → 补牌友善开启？ │ yes → 牌堆顶多看N×3张，按匹配度排序取前N张 │ no → 正常补牌 │ └─ 结算 → 安全阀检查 └─ END
