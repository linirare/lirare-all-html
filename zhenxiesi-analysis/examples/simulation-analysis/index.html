Summary

四大关键趋势

① 合成取代三消

合成正在取代三消作为经营游戏的"动作层"。微信小游戏用户比App用户更轻度，合成结果可控、卡关感弱。过去4年新增的模拟经营小游戏中，合成+经营占比超过一半。

② 越土越赚钱

动物餐厅（动物）、乌冬旅店（治愈）、李国富（普通人）、房东模拟器（收租）——越是接地气、有情绪共鸣的题材，裂变越好。小游戏端尤其明显。

③ 国风APP壁垒

江南百景图、桃源深处有人家、这城有良田——中国古风美术在App端仍是最稳的题材选择，但在小游戏端表现一般。

④ 小游戏独立品类

微信小游戏已不再是"App降级版"，而是有自己的核心品类（合成+经营、开箱+经营）和成功方法论。240+款小游戏季度流水超千万。

4 key trends
这四个趋势涵盖了品类创新、题材选择、平台差异和市场格局，是做模拟经营必须理解的方向。
