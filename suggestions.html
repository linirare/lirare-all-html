🔴 P0 · 紧急
#2 成长节奏
P0元婴太快、化神太慢 ❌ 未解决 建议：降低元婴门槛→提高元婴至化神过渡平滑度。保持第一天可达元婴的爽感，同时在元婴→化神间增设中期目标与资源投放；化神所需数值大幅降低或增加"突破任务线"替代纯堆数值。 💬 玩家说："第一天整好了能上元婴""化神太难了""化神300级需要太多升不上去"
#3 国家差异
P0第三天国家差异大，国战崩 ❌ 未解决 建议：新服选国时可视化各国人数与活跃度，引导均衡分布；赛季结束重置时允许换国；建立弱势国补偿机制（资源加成/保护期），防止一边倒导致大面积弃坑。 💬 玩家说："三个国家实力差距太大了""这国战没选唐的都麻了""根本抢不过大唐的 我大理就两三个在线的"
#6 打架奖励
P0战场打架打不起来，奖励不吸引 ❌ 未解决 建议：BOSS战奖励机制重做——死亡/非首伤也应获得部分奖励，按伤害排名给予保底。国战等大型活动奖励大幅提升，与投入时间匹配。增加助战奖励机制。 💬 玩家说："打了半天死了奖励都没有""没奖励都不愿意打""国战一个月第一只给2000玉""帮人家打没奖励"
#7 令牌归属
P0修士历练牌子难消化 ❌ 未解决 建议：增加历练令牌非充值获取途径（活跃/宗门/周常等）；归属方产出*2、非归属方产出*1 保底，避免非归属国家玩家"打了白打"。令牌商店增加可兑换内容。 💬 玩家说："令牌除了兑换和任务没有能买的地方""不是伤害第一就没得拿""没令牌了帮人打啥奖励都没"
#11 战场过多
P0同时战场过多，打不起来 ❌ 未解决 建议：合并低参与度战场/活动，减少同时开放数量，将玩家热度集中在有限战场中。避免"国战一个月"式的长时间低热度活动。 💬 玩家说："国战还是少些好""主国战估计人都要散光""没人打元婴BOSS全是大唐的"
🟠 P1 · 重要
#8 集体感差
P1国家人数过多，集体感差 ❌ 未解决 建议：国家内部按活跃度/战力分组编队，设立小队长机制；国家频道增加内置语音/快捷指挥功能；国家任务与国家进度绑定，增强共同目标感。 💬 玩家说："大唐是不是人最多的""大理就两三个在线的""根本抢不过"
#9 结拜系统
P1增加结拜，制造小圈子 ❌ 未解决 建议：3-5人结拜系统——共享进度任务、独立聊天频道、结拜技能/加成、结拜标识。满足玩家找固定队友、建立小圈子的社交需求。 💬 玩家说："我们提前联盟一起发育""3区有组队没""组队干什么？可以组队？""先加我好友"
#12 抢资源
P1战场增加抢资源，才能活起来 ❌ 未解决 建议：战场地图中投放限时修炼资源点/灵石矿脉/丹药材料，玩家/国家争夺采集权。打架有看得见的收益，而非纯粹为了排名积分打。 💬 玩家说："资源都被抢完了不叛怎么活""不抢资源直接干啊""氪佬不在你国家吊起来抽资源拿不到"
#5 御兽刷新
P1御兽每日 0 点刷 ❌ 未解决 建议：所有每日刷新机制（御兽/商店/任务/活动）统一为 0 点，避免玩家混淆不同刷新时间点。增加刷新倒计时显示。 💬 玩家直接要求："御兽能不能改成统一0点刷新"。同时多人问"什么时候刷新啊"答"0点"说明信息不透明
#13 联盟管理
P1人太多没人管理，需联盟增强前期交互 ❌ 未解决 建议：增加"联盟"组织——在国家之下的次级团体，由玩家自主建立和管理。联盟拥有独立领地、建设任务、联盟BOSS。解决"几百人一个国家没人管"的问题。 💬 玩家说："管理呢""我们提前联盟一起发育""确实一区热闹人多才好玩"但国家内部却一盘散沙
🔵 P2 · 改善
#1 首日操作
P2首日点的太多，操作繁琐 ✅ 部分已修 建议：基础操作（采集/传道/升级）默认支持十连/一键，非月卡玩家也应开放。减少一块一块采、一级一级点的机械操作。
5/29 已更新默认十次+月卡一键，但非月卡仍需手动十次，仍有优化空间。 💬 玩家说："筑基丹材料难打人物升级按的次数太多""十年采集要月卡确实离谱很劝退""我一个元婴老祖只能一块一块采"
#4 衙门布局
P2衙门布局操作不顺畅 💡 建议 建议：重构衙门界面操作路径，高频功能前置，减少页面跳转层级。具体策略需结合实际 UI 走查。 玩家未集中反馈此问题，推测为个人体验痛点
#10 官品兑换
