03FARMING

FARMING SYSTEM

种花系统 · 类似QQ农场但更深

🌾 花卉种类

  • 100+种花卉：玫瑰/郁金香/百合/绣球/康乃馨等
  • 分：普通花种、珍稀花种、限定花种（活动获取）
  • 不同品种：生长周期不同、售价不同、插花配方不同
  • 种子获取：商店购买/任务奖励/活动/好友赠送

⏱️ 生长周期设计

  • 短期花（5-15分钟）：基础花种，新手过渡
  • 中期花（30-60分钟）：主力经济花种
  • 长期花（2-8小时）：高收益花种
  • 稀有花（12-24小时）：限时/活动花种
  • 设计目的：覆盖不同登录频次的玩家

💧 种植操作

  • 开垦土地：消耗金币解锁新地块
  • 播种：从种子包选择→种植到地块
  • 浇水：拖拽水壶到土地上，可一次性浇多块
  • 收获：成熟后点击收取
  • 进阶：施肥（加速生长）、除虫（防减产）

📈 土地扩张节奏

初始6块地，随等级逐步解锁。每开一块地消耗递增金币/钻石。土地上限约30-50块。扩张驱动：可以同时种更多花→产出更多→赚更快

100+花卉 · 四级生长周期 · 地块扩张驱动
04ORDERS

ORDER & SHOP SYSTEM

接单经营 · 金币产出的发动机

📋 订单系统

  • NPC定时发布花束订单
  • 订单要求：指定花种+数量+品质
  • 完成订单获得金币+经验+随机奖励
  • VIP订单（高难度）→ 更高收益
  • 限时订单 → 紧迫感驱动

💐 插花系统

  • 将不同花卉组合成插花作品
  • 插花配方：不同搭配→不同作品
  • 上架到花坊 → 等待客人购买
  • 插花有品质评分 → 影响售价

🏪 花坊经营

  • 花坊等级：决定可接订单的等级
  • 花坊装饰：吸引更多/更高级客人
  • 雇佣助手：自动完成部分操作
  • 花坊扩建：更多展示位+生产位

⚙️ 经营节奏设计

种花（供给端）和订单（需求端）的平衡：种了卖不掉=浪费,订单完不成=损失收益。引导玩家规划种植品种和数量,产生"策略感"

订单+插花双线产出 · 供需平衡策略

06MONETIZATION

MONETIZATION MODEL

三层付费 + 创新实物变现

📺 广告变现

  • 浇水加速 → 看广告缩短生长时间
  • 订单刷新 → 看广告重置订单列表
  • 额外奖励 → 每日广告宝箱
  • 免费种子 → 看广告领稀有种子

💎 内购系统

  • 钻石（充值币）：购买种子/加速/装饰等
  • 月卡：每日领取钻石+加速特权
  • 礼包：新手礼包/节日礼包/等级礼包
  • 限定装饰：氪金专属花园主题

🌹 创新变现：实物花束

  • 核心营销语："线上种花，线下收真花"
  • 游戏内达成条件→兑换真实花束→快递到家
  • 高额充值可得"真花礼盒"
  • ⚠️ 争议：大量用户投诉收到花束与宣传不符 / 未收到 / 客服不处理
  • 被多个媒体曝光虚假宣传问题

💰 付费触发时机

  • 开地缺金币 → 推送金币礼包
  • 种花等太久 → 推送加速道具
  • 限时活动 → FOMO驱动消费
  • 差一点完成订单 → 推送所需花卉包
广告+内购+实物三线 · 实物变现存争议
07MARKETING

GO-TO-MARKET STRATEGY

明星代言+实物钩子+饱和投放

⭐ 明星营销

  • 代言人：杨紫（国民度+女性粉丝群体契合）
  • 代言画面：杨紫手持鲜花，传递"治愈"感
  • 院线电影贴片广告（2026春节档）
  • 抖音/B站/小红书种草投放

🌹 实物钩子

  • "种花收真花"=极强的差异化卖点
  • 社交传播力强——收到花拍照发小红书
  • 相比纯游戏赠送礼物，实物成本可控
  • 但履约是最大挑战

📈 投放策略

  • 字节系（抖音/头条）为主投渠道
  • 素材方向：种花治愈过程/花坊装扮展示/收到真花惊喜
  • 女性向精准定向：25-40岁+兴趣标签+花艺
  • CPA预估：10-20元/下载

🏆 成为黑马的原因

  • 品类选择精准：治愈经营+女性向=大市场
  • 实物钩子制造传播爆点
  • 明星代言拉升品牌信任
  • 买量素材抓住情感诉求（治愈/放松）
  • 上线时机好（暑期档+春节档连续引爆）
IP代言+实物钩子+情感投放=出圈公式
08PSYCHOLOGY

PLAYER PSYCHOLOGY

治愈系游戏的心理学机制

🧠 种花的心理机制

  • 预期的快乐：播种后期待绽放的过程 > 收获本身的快乐
  • 责任感：每天回来浇水照看 = 养成习惯
  • 治愈感：绿色植物+花朵 = 天然的减压元素
  • 进度可视化：从种子到花开的完整生命历程

📦 经营的成就感

  • 从小到大的成长：从几块地到大型花坊
  • 策略规划：种什么花、接什么单、怎么布局
  • 经济系统正反馈：投入→产出→扩张→更多投入

🎨 装扮的展示欲

  • 主权感："我的花园我做主"→投入情感
  • 社交展示：好友参观→获得点赞→继续投入
  • 稀缺性：限定装饰→收集欲望

🔗 沉没成本锁住用户

  • 每天浇水照顾→形成习惯依赖
  • 花坊装扮投入→不可逆→舍不得走
  • 收集的花种/图鉴→想集齐
  • 好友互助关系→社交绑定
治愈+养成+展示+归属 = 女性向核心驱动力

10COMPETITION

COMPETITIVE LANDSCAPE

模拟经营+种花赛道的玩家

🌾 品类同源：QQ农场

2009年现象级社交游戏，核心玩法种菜→收菜→偷菜。我的花园世界可以理解为"QQ农场的现代升级版"——种花代替种菜、花店代替仓库、更多装扮和经营深度

🌻 直接竞品：花店经营类

  • 《 flower shop 》系列（海外）
  • 各类"花店模拟器"小游戏
  • 樱花校园模拟器（部分玩法重叠）
  • 但这些产品品质和投入远不如本作

📊 差异化优势

  • 明星代言带来的品牌信任感
  • 实物花束的差异化钩子
  • 高品质美术（1.59G大包体=高品质信号）
  • 社会话题性（真花争议反而带来讨论度）

⚠️ 潜在风险

  • 品类壁垒低，易被复制
  • 实物履约成本高，争议影响口碑
  • 内容消耗快，需持续更新
  • 买量依赖重，CPI持续上涨
QQ农场的现代升级版 · 实物变现是双刃剑
11CONTROVERSY

CONTROVERSY & RISKS

"真花"承诺引发的信任危机

⚠️ 事件概述

2026年4月，多家媒体（西西新闻等）曝光《我的花园世界》"线上种花线下收真花"涉嫌虚假宣传。消费者反映充值后未收到花束/收到的花束与宣传不符。客服回应迟缓，投诉难以解决

📋 问题点

  • 充值门槛与宣传不符
  • 花束质量差（枯萎/品种不对）
  • 物流缓慢/根本不发货
  • 客服联系不上/推诿
  • 黑猫投诉平台大量集体投诉

📊 影响分析

  • 短期：话题曝光带来下载量↑
  • 中期：负面口碑影响付费转化↓
  • 长期：信任被透支→用户流失
  • 监管风险：可能被市监局约谈/处罚

🎯 给我们的教训

  • 实物挂钩很有效，但履约是生死线
  • 营销承诺必须≤实际交付能力
  • 客服体系需要提前建设好
  • 口碑=长期生命线，不能透支
营销是把双刃剑 · 口碑是长期生命线
12TAKEAWAYS

KEY TAKEAWAYS

给我的花园世界的启示

1️⃣ 品类选择 > 玩法创新

种花经营不是新玩法，但它选择了"治愈感+女性向"这个巨大且被验证的市场。有时候不需要发明新玩法，把成熟玩法做到极致+对的用户群+对的营销就够了

2️⃣ 实物钩子 = 双刃剑

"线上种花线下收花"是天才级的营销创意，产生了极强的传播力和下载转换。但如果没有稳定的供应链和客服体系支撑，信任崩塌后反噬更严重

3️⃣ 明星代言是放大器不是发动机

杨紫代言拉高了品牌格调，但如果产品本身的核心体验留不住人，代言只是加速了用户的流逝。产品力+营销力必须同时在线

4️⃣ 模拟经营的"每日回访"是最强留存

种花有生长周期 → 每天要回来浇水收获 → 形成习惯。比三消类的"想玩才玩"更稳定。这种"不得不回来"的机制是模拟经营品类的先天优势
